##HOW THE ORIGINAL DATA HAS BEEN MODIFIED?
1 - MERGING and TRAINING the test sets to create a unique dataset
2 - Extracting, to each measurement, the measurements on the MEAN and the STD
3 - Using activity names to name activities in the dataset
4 - Labeing the data with appropriate descriptive names
5 - Creating a second tidy dataset with the avg of each variable for each activity and subject
